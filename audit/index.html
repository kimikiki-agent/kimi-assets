🔗 依賴

RiveGen (Phase 2/3 動畫序列化) → AvatarController (情緒映射與動畫混合)
RiveGen 的 Phase 2/3 完成後，AvatarController 的 PriorityMixer 才能正確序列化複雜的動畫參數疊加。目前 AvatarController 的 [todo] PriorityMixer.ts 等待 RiveGen 的 Animation/State Machine 支援

persona/skill-pack 的獨立配置機制需要依賴 AgentToolkit 的 agent-init script 進行初始化，以及 skill-sync 進行技能包同步。目前 AgentToolkit 的相關功能已實作但 checklist 未更新
SessionHandoff (auto-init-before-build) → 所有新專案初始化流程
auto-init-before-build 完成後會簡化新專案建立。目前 Katy 的工作流需要手動執行 init，新機制應自動檢查並補齊缺失的 CLAUDE.md、plans/、claude-status.json
